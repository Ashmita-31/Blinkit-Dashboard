# 🛒 Blinkit Sales Dashboard

## 📌 Project Overview

The Blinkit Sales Dashboard is an interactive data analytics project built using Microsoft Excel. The dashboard analyzes Blinkit's sales data and presents meaningful business insights through KPIs, Pivot Tables, Charts, and Slicers. It enables users to explore sales performance, product trends, and customer purchasing patterns in an easy-to-understand format.

---

## 🎯 Objectives

The main objectives of this project are:

- Analyze overall sales performance.
- Track important business KPIs.
- Compare sales across product categories.
- Identify high-performing products.
- Understand customer buying trends.
- Build an interactive dashboard for better decision-making.

---

## 📂 Dataset

The dataset contains sales information such as:

- Product Name
